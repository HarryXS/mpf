mpf.platform.snux module

Contains the base class for the Snux driver overlay.

This class overlays an existing WPC-compatible platform interface to work with Mark Sunnucks’s System 11 interface board.

driver_action(driver, milliseconds)

Adds a driver action for a switched driver to the queue (for either the A-side or C-side queue).

Parameters:
  • driver – A reference to the original platform class Driver instance.
  • milliseconds – Integer of the number of milliseconds this action is for. 0 = pulse, -1 = enable (hold), any other value is a timed action (either pulse or long_pulse)

This action will be serviced immediately if it can, or ASAP otherwise.
